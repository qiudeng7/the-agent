## 5. 核心用户流程

```
用户进入
   │
   ├──→ 首页打招呼 → 快捷卡片/输入框 → 发送问题 → AI 回复
   │                                        │
   │                                        └──→ 继续追问 / 新建会话
   │
   ├──→ 左侧边栏
   │     │
   │     ├──→ 点击历史会话 → 加载对话上下文
   │     ├──→ 搜索框 → 搜索历史/应用
   │     └──→ 全部分组 → 查看/创建新会话
   │
   └──→ 全部应用
         │
         ├──→ 浏览分类 (官方推荐/工具/娱乐...)
         ├──→ 点击 AI 应用 → 进入专用界面
         └──→ 创建智能体 → 自定义 AI 助手
```

## 6. 关键交互特点

| 特点 | 说明 |
|------|------|
| **会话管理** | 左侧聊天历史，点击即切换，支持分组 |
| **快捷入口** | 首页显示推荐问题/功能卡片 |
| **多模型支持** | 顶部可选择 DeepSeek 等不同模型 |
| **功能开关** | 深度思考、联网搜索、工具扩展 |
| **应用生态** | 应用广场提供垂直场景的 AI 助手 |

## 7. 可借鉴到我们的 Agent 项目

### 路由设计

```typescript
// src/router/index.ts
const routes = [
  { path: '/', name: 'home', component: Home },      // 首页/欢迎页
  { path: '/chat/:id', name: 'chat', component: Chat }, // 会话详情
  { path: '/apps', name: 'apps', component: Apps },     // 应用广场
  { path: '/settings', name: 'settings', component: Settings }, // 设置
]
```
