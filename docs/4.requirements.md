# 需求清单

## 一、项目基础

- [x] Electron + Vite + Vue 3 + TypeScript 项目骨架
- [x] Electron Forge 打包工具（替换 electron-builder）
- [x] Windows 跨平台构建命令支持
- [x] 生产构建白屏问题修复（asar 路径）
- [x] 应用启动 loading 动画

## 二、开发工具链

- [x] Chrome DevTools MCP 集成（端口 9223）
- [x] Electron 远程调试端口在 `app.whenReady()` 前正确注册
- [x] Gitea 自动发布 skill（保留最近 5 个版本）
- [x] Claude Agent SDK 文档

## 三、界面设计系统

- [x] Organic 风格设计语言
- [x] 设计 token：颜色、字体、圆角、阴影、过渡动画（CSS 变量）
- [x] 深色模式 CSS 变量（`.dark` 类）
- [x] 所有组件深色模式适配（背景、边框、文字均使用变量）
- [x] `App.vue` 监听设置状态动态切换 `.dark` 类

## 四、窗口与标题栏

- [x] 无边框窗口（`frame: false`）
- [x] macOS 保留系统原生三个点按钮（`titleBarStyle: 'hiddenInset'`）
- [x] 顶部透明拖拽区域（32px，`-webkit-app-region: drag`，绝对定位不遮挡内容）
- [x] 窗口控制按钮（`WindowControls.vue`）绝对定位，不随侧边栏状态变化
  - [x] macOS：位于三个点右侧（`left: 72px`）
  - [x] Windows：贴靠左上角（`left: 8px`）
- [x] IPC：`closeWindow` / `minimizeWindow` / `maximizeWindow`

## 五、侧边栏

- [x] 搜索会话输入框（与 chat store 双向绑定）
- [x] 导航菜单：首页、全部应用
- [x] 会话分组列表（`SessionList.vue`）
- [x] 新建分组（弹窗输入名称）
- [x] 点击会话跳转对应对话页
- [x] 用户区域固定在底部（独立于滚动区域）
- [x] 点击用户区域跳转设置页
- [x] 侧边栏收起 / 展开（`collapsed` prop 由父级 `AppLayout` 管理）
- [x] 对话历史标题溢出修复（`text-overflow: ellipsis`）
- [x] 侧边栏整体加 `-webkit-app-region: no-drag` 防止 drag region 拦截点击

## 六、主要页面

### 首页（`/`）
- [x] 欢迎语（用户姓名）
- [x] 快捷功能卡片（创作文字、生成图片、分析问题、编程帮助）
- [x] 建议问题列表
- [x] 对话输入框（发消息后跳转新对话页）

### 对话页（`/chat/:id`）
- [x] 消息列表（用户消息 / AI 回复区分样式）
- [x] `ChatInput` 组件：深度思考开关、联网搜索开关、工具按钮、模型选择
- [x] 新建会话按钮跳转首页（不立即创建会话记录）

### 应用广场（`/apps`）
- [x] 应用卡片展示
- [x] 分类筛选标签

### 设置页（`/settings`）
- [x] 语言选择：跟随系统 / 中文 / 日本語 / English
- [x] 深色模式：跟随系统 / 深色 / 亮色
- [x] 系统语言自动检测，非中日英系统默认 English
- [x] 设置持久化（`localStorage`）
- [x] 监听系统主题变化（`prefers-color-scheme`）

## 七、状态管理

- [x] `useChatStore`：会话列表、分组、当前会话、搜索关键词
- [x] `useSettingsStore`：语言、主题、持久化、系统主题监听
- [x] `useAgentStore`（基础结构）
